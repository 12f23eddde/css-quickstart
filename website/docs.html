Install CLI

Requirements

  • OS
    Windows 10 x64 1507, Mac OS X 10.12, Linux 4.15 or newer
  • Node
    Node.js 12.x or newer

Install

Via homebrew: 

$ brew install tsugu-cli

Via npm: 

$ npm install -g tsugu-cli

Via yarn: 

$ yarn install -g tsugu-cli
